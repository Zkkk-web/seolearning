What is SEO?

Search Engine Optimization
Google dominates the search engine market,
and when we talk about search engine optimization,
most cases we are actually referring to the optimization for Google.

"The essence of SEO is:
to enable search engines to better understand your web pages"

Theory

Webmasters and search engines

The relationship of ecological complementarity

Why does Google index so many web pages?

In order to meet the search needs of Google users and maintain its leading position in the search engine market

Why do webmasters create so many web pages?

Because there is user demand, because I want to make money, because I want to prove myself, and because I just want to contribute to others.

Google pursues two goals

  1. Newest:Meet the constantly emerging new search demands
  2. The most comprehensive: Meeting all kinds of strange and curious needs of every user

The goal of the crawler system

Crawl all high-quality web pages across the entire web

Indexes

  • Positive index: What keywords does a certain web page have
  • Inverted index: Which web pages are included for a certain keyword

How to do a good SEO

Two goals

  1. It is crawled by web crawlers and enters the index database of search engines
  2. Enter the first two pages (top 20) of the search results

In order to be crawled by web crawlers, we need to do a good job in in-site SEO.
To secure a top position in search results, we need to enhance the website's weight, web page weight, and user experience

Three practical SEO experiences

  1. On-Page SEO helps our web pages enter the search ranking range
  2. External links help us enter the top 10 positions in Google search results rankings
  3. The user experience of the website determines the final ranking position

SEO work is divided into two parts:
on-site and off-site

  1. instation:Do a good job in planning the web page structure of the website, and do a good job in the internal link structure between web pages, so that search engines can more easily crawl our web pages, more easily understand our web pages, and more easily extract the theme and best functions of each web page, meet user needs, and improve user experience
  2. Outside the site, we need to do a good job in publicity and promotion, so that our website brand can be mentioned by many other websites' web pages and our web page links can appear on many other websites

practice is the sole criterion for testing truth
